Protocolos de rede

O que são protocolos de rede?

Na ciência da computação, um protocolo é um padrão (conjunto de regras convencionadas) que controla e possibilita a conexão, comunicação e, transferência de dados entre dois sistemas computacionais, ou seja, em uma rede de computadores o protocolo é o responsável pela comunicação entre os computadores (a linguagem padrão de comunicação) e também responsável pela troca de dados entre eles, independente da localização e independente do sistema operacional/plataforma utilizada nesses equipamentos. De maneira simples, um protocolo pode ser definido como "as regras que governam" a sintaxe, semântica e sincronização da comunicação. Os protocolos podem ser implementados pelo hardware, software ou por uma combinação dos dois.

Camadas de rede

A camada de rede do modelo OSI lida com as transferências de pacotes de uma origem para um destino. O seu papel é mais sofisticado que a camada de enlace, pois em vez de de transferir quadros de uma extremidade até a outra de um fio, a camada de rede precisa dar hops de roteador para roteador.
